# Node.js Auth CRUD API 🚀

## 📌 About
A simple Node.js REST API with:
- User registration & login (JWT)
- Protected CRUD tasks
- MongoDB

## ✅ Tech Stack
- Node.js
- Express
- MongoDB (Mongoose)
- JWT, bcrypt
